# R223M-P3 Screen Trigger Standard

大屏内容必须从课堂事件中长出来，不得脱离任务、偏差、示范、关键词或证据提交。

Required backend fields: `type`, `content`, `source_event_moment`, `screen_prompt`.

| Event | Screen Type | Content | source_event_moment | Prompt |
| --- | --- | --- | --- | --- |
| 忆一忆：毛笔的诞生 | `structure_recall` | 毛笔工序图 + 笔头/笔杆/笔尾圈画 | 任务释放和学生说不清工序用途时 | 这一步让毛笔哪里更好用？ |
| 探一探：关键的结构 / 十万个为什么 | `question_cluster` | 四个为什么 + 结构圈画 + 关键词 | 同桌猜想后和答案收束时 | 这个小结构解决了什么使用问题？ |
| 设计会：像设计师一样思考 | `design_case_compare` | 设计案例细节圈画 + 握/放/收/找方向 | 学生从观察转入改进想法时 | 这个小巧思帮谁解决了什么麻烦？ |
| 试一试：1+1 合作小设计 | `student_work_compare` | 一件只是装饰的作品 + 一件解决防滑/防滚/易找问题的作品 | 学生制作 4 到 5 分钟后 | 哪一件解决了一个真实使用问题？ |
| 创一创：1+n 文具大变身 | `technique_and_material_demo` | 盘的技法、线材/纸材/形状材料半成品、材料过多与合适材料对比 | 从 1+1 迁移到 1+n、学生不会固定材料或材料堆叠时 | 这个材料的哪一点帮你解决了问题？ |
| 笔友汇 | `keyword_feedback` | 结构、功能、材料、使用、方便、牢固、舒服、有创意词云 | 作品展示和同伴点评时 | 请选一个关键词，说说它哪里有巧思。 |
| 赠笔礼 | `closing_sentence_and_unit_task` | 赠笔语句式 + 《一年级购买文具建议书》《文具课堂使用指南》 | 赠笔表达和全课收束时 | 我把它送给……因为它可以…… |
